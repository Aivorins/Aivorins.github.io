HOW IT WORKS

Three layers of protection.

ACB runs three checks in sequence — from fastest to most thorough. Most blocked sites never load a single byte of content.

01 · NETWORK
Domain Filter
30,000+ known adult domains are blocked at the network level using Chrome's Declarative Net Request API. The page never loads. No delay, no flash of content.
02 · URL
URL Pattern Check
The hostname and path are checked against 30+ keyword patterns. Sites with explicit signals in the address itself are caught and redirected before any page content loads.
03 · CONTENT
Page Content Scan
If a page passes the first two checks, its text is scored across five signal categories — explicit acts, adult content, meta tags, and more. High-scoring pages are blocked and cached for 30 days.

FEATURES

Everything it does.

Password-Protected Toggle
Turning the extension off requires the password you set when activating it. The hash is stored locally with SHA-256 — never readable, never sent anywhere.
Site Whitelist
Add any domain to the whitelist and it will never be scanned or blocked. Medical, educational, and mainstream sites are pre-whitelisted out of the box.
30-Day Reputation Cache
Sites flagged by the content scanner are remembered for 30 days. If you return, the block is instant — no re-scan needed.
Calm Blocked Page
When a site is blocked, users see a peaceful page — soft design, a motivational quote, and ambient music. Not a jarring red warning screen.
RTA & ICRA Meta Tag Detection
Sites that self-label as adult content using RTA-5042 or PICS-label meta tags are caught immediately — before the content is even read.
Parental Control Ready
Set a password at install time and the extension activates. Disabling it requires the password — making it a meaningful barrier for parental oversight.
